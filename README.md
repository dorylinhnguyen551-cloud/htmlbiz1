# Delish Bites — Local Business Website

Simple multi-page static site (HTML/CSS/JS) created for a local restaurant. Files:

- `index.html` — Home page with slideshow and best-sellers
- `menu.html` — Menu page (drinks first, then food)
- `locations.html` — Map and nearby location suggestion
- `catering.html` — Catering contact form
- `styles.css` — Styles (dark blue background, white text, Oswald title font)
- `script.js` — Client JavaScript for slideshow, geolocation, and forms

To preview locally:

```bash
# from this folder
open index.html
# or run a simple static server (python 3)
python3 -m http.server 8000
# then open http://localhost:8000
```
